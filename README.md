#
This project is a Java program that checks whether two strings are anagrams.
An anagram means both strings contain the same characters in a different order (e.g., listen and silent).

#
How It Works:

1)Takes two strings as input from the user.

2)Converts both strings to lowercase and removes spaces.

3)Converts the strings into character arrays.

4)Sorts the arrays and compares them.

5)If both arrays are equal, the strings are anagrams; otherwise, they are not anagrams.
